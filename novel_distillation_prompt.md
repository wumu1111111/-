# 🔥 长篇网文蒸馏提示词系统 v1.0

> 用于从百万字网文中提取核心信息，为同人创作提供精确参考

---

## 📌 使用说明

**这个提示词系统分为两个阶段：**
1. **第一阶段**：全文快速蒸馏（概览全局）
2. **第二阶段**：细节深挖（精准还原）

根据你的网文字数和LLM容量选择：
- 字数 < 50万：全部一次性处理
- 字数 50-200万：分段处理（每50万字一段）
- 字数 > 200万：优先提取核心章节（开篇、重要转折、结尾各20%）

---

## 🎯 第一阶段：全文蒸馏框架

**这个阶段用来快速理解整部作品的骨架**

```
你是一个专业的网络文学编辑和蒸馏师。
现在有一部长篇网文需要你进行深度蒸馏提取。

请你按照以下框架，对这部作品进行结构化分析和蒸馏。
这份蒸馏内容将用于同人创作，需要保证：
✓ 准确还原原作的核心设定和风格
✓ 保留所有重要的剧情逻辑和伏笔
✓ 重现作品的文笔特色和情感基调
✓ 提供足够的细节让续写者无缝接轨

【第一部分：作品基本信息】
◆ 作品分类：[网络文学类型 - 例：网游/悬疑/恋爱/克苏鲁等]
◆ 故事背景：[时代/地点/世界观 - 例：21世纪末地球/异世界大陆等]
◆ 核心概念：[这部作品最核心的设定是什么，一句话概括]
◆ 文笔风格：[作者的语言特点 - 例：细腻抒情/简洁直白/文言风等]
◆ 情感基调：[作品的主要氛围 - 例：温馨日常/压抑诡异/热血燃烧等]
◆ 总体框架：[从开始到结束，故事的大致走向]

【第二部分：人物系统 - 核心角色档案库】
对于每个重要人物（主角、主要配角、关键反派），建立详细档案：

[角色 #1 - 主角或第一主人公]
• 基本信息：名字 | 性别 | 年龄/外表年龄 | 身份
• 首次出场描写：[原文对这个角色的第一印象描写，保留原文风格和措辞]
• 外貌特征：[详细的视觉特征，包括气质、装扮、标志性特征]
• 性格核心：[三个关键词概括核心性格]
• 性格细节：
  - 说话习惯：[特殊的说话方式、口头禅、用词特点]
  - 行为习惯：[经常做的事、习惯姿态等]
  - 思维特点：[思考方式、逻辑风格]
• 能力系统：[详细列出所有技能、天赋、特殊能力]
• 装备/持物：[经常携带的物品、武器、身份象征物，要包括来源故事]
• 背景故事：[家世、成长经历、重要经历，按时间线详细说明]
• 目标与动机：[这个角色在故事中想要什么、为什么要这样做]
• 人物弧线：
  - 开局状态：[故事开始时的心态和处境]
  - 关键转变点：[在哪些事件后性格/观念发生了改变]
  - 最终蜕变：[到故事结束时这个角色变成了什么样]
• 与其他角色关系：[对主要配角的看法、互动模式、感情发展]
• 经典台词或行为：[最能代表这个角色的台词或行为]
• 读者的情感连接：[这个角色吸引读者的地方是什么]

[角色 #2-N - 重要配角和反派]
（按相同格式，但篇幅可适当简化）

【第三部分：世界设定系统】
◆ 地理环境：
  - 主要舞台：[故事主要发生的地点，包括具体描写]
  - 重要地点：[其他关键地点及其特色，与主线的关联]
  - 地理特色：[这个世界的地理如何影响故事]

◆ 权力结构与组织：
  - 统治者/政治结构：[谁在统治，权力如何分布]
  - 重要组织：[涉及的主要势力、家族、帮派等，及其立场]
  - 社会阶级：[存在的阶级分化或身份系统]

◆ 规则系统（核心设定）：
  如果是有超能力/魔法/修炼的世界，详细说明规则：
  - 能力名称：[如"灵气""异能""魔法"等]
  - 分级体系：[如何划分等级、具体标准]
  - 获得方式：[如何获得这种能力]
  - 使用限制：[有什么限制条件]
  - 具体表现：[实际使用时的具体效果]
  - 重要术语：[与这个系统相关的专业名词及其含义]

◆ 文化与常识：
  - 独特文化：[这个世界特有的文化风俗、节日等]
  - 常见价值观：[这个世界普遍认同什么价值观]
  - 语言特色：[如果有特殊用语、方言等要标注]

◆ 时间设定：
  - 故事时间跨度：[从第一章到最后一章相隔多长时间]
  - 重要时间点：[关键事件发生在什么时候]
  - 时间流速：[故事的叙述时间和实际时间是否有差异]

【第四部分：剧情骨架系统】
◆ 总体故事结构：
  - 序章/开篇：[故事如何开始，第一个引入点是什么]
  - 第一阶段（第1-？章）：
    * 主要事件：[按时间顺序列出重要事件]
    * 冲突点：[在这个阶段遇到了什么主要问题]
    * 结束于：[这个阶段以什么事件作为结尾]
  - 第二阶段（第？-？章）：
    （同上格式）
  - 第三阶段（第？-？章）：
    （同上格式）
  - ... 后续阶段依次列出
  - 结局：[故事如何结束，最终冲突如何解决]

◆ 主线剧情走向：
  [简要描述主角从第一章到最后一章的经历路线]

◆ 重要伏笔与解开：
  - 伏笔 #1：[什么是伏笔，在第几章埋下，第几章解开，如何解开]
  - 伏笔 #2：... 以此类推

◆ 关键转折点：[列举3-5个最重要的剧情转折，说明转折前后的变化]

◆ 节奏特点：
  - 高潮位置：[故事的最高潮在哪里]
  - 节奏起伏：[故事是快速推进还是缓慢铺垫，在什么地方变化]

【第五部分：核心主题与风格】
◆ 核心主题：[这部作品想表达什么核心思想或主题]
◆ 情感弧线：[读者的情感如何变化 - 从什么心情变到什么心情]
◆ 最独特之处：[这部作品相比同类型作品的最大优势是什么]
◆ 标志性元素：[这部作品有什么标志性的元素，让人一眼就认出来]
◆ 作者笔法特色：
  - 描写风格：[对话/环境/心理/行动的描写习惯]
  - 语言特色：[常用的修辞手法、成语、俗语等]
  - 节奏把握：[短句/长句的使用习惯]
  - 情感表达：[如何表达人物的内心活动]

【第六部分：续写注意要点】
◆ 剧情逻辑链条：[确保续写时剧情因果关系要遵循]
◆ 人物性格底线：[各个角色性格中不能改变的核心特征]
◆ 设定约束条件：[世界设定中有什么硬性规则不能违反]
◆ 读者心理预期：[读者对后续故事最可能的期待是什么]
◆ 风格保持方法：[如何在续写中保持与原文一致的风格]

```

## 🎬 第二阶段：细节深挖提示词

**当你需要更深入了解某个人物、某个场景或某条线索时使用**

```
请对以下内容进行深度分析：

【深度人物分析】
针对 [人物名字]，我需要你：

1. 心理建设：
   - 这个人物的内心世界是怎样的？
   - 表面行为和真实想法之间有什么差异？
   - 这个人物最害怕什么，最渴望什么？

2. 言行一致性：
   - 列举这个角色的典型台词（原文）
   - 这个角色的行为方式有什么规律？
   - 在什么情况下会打破常规行动？

3. 人物成长：
   - 从开篇到现在，这个角色思想观念改变了什么？
   - 是什么经历导致了这些改变？
   - 还有哪些潜在的成长空间？

4. 人际互动风格：
   - 这个角色如何对待不同的人（主角/敌人/陌生人）？
   - 有什么特殊的人际模式？
   - 与其他角色互动时的独特之处？

【深度场景还原】
针对 [具体场景或章节]，我需要你：

1. 场景氛围：
   - 原文如何描写这个场景的环境氛围？
   - 这个场景的情感基调是什么？
   - 有什么细节描写让这个场景特别？

2. 事件逻辑：
   - 为什么会发生这个事件？
   - 涉及哪些角色，每个角色的动机是什么？
   - 这个事件如何推动了故事发展？

3. 转折意义：
   - 这个场景之前故事是什么样的？
   - 这个场景改变了什么？
   - 对后续故事的影响是什么？

【深度设定解读】
针对 [具体设定或规则]，我需要你：

1. 设定细节：
   - 这个设定在原文中如何被首次引入？
   - 具体规则是什么？有什么例外情况？
   - 原作者如何通过故事展示这个设定？

2. 实际应用：
   - 在故事中如何实际使用或体现这个设定？
   - 有哪些具体的例子？
   - 这个设定如何影响角色的行为？

3. 逻辑一致性：
   - 这个设定与其他设定之间的关系是什么？
   - 有没有逻辑漏洞或矛盾之处？
   - 原作者如何解释或圆回可能的疑点？
```

## 💻 实施步骤

### 步骤1：文本分割（如果超过150万字）
```
将网文按以下比例分段：
- 开篇精选部分（第1-100章或前20%）
- 中间关键章节（重要转折点所在的章节）
- 结尾精选部分（最后20%的章节）
- 任何你认为关键但可能被忽略的章节

每段文本应该是5000-15000字的完整内容。
```

### 步骤2：分阶段处理
```
第一轮：用第一阶段提示词处理开篇部分 → 获得基础框架
第二轮：用第一阶段提示词处理中间关键部分 → 补充情节信息
第三轮：用第一阶段提示词处理结尾部分 → 完善故事全景
第四轮：用第二阶段提示词对关键人物/场景进行深挖 → 精化细节
```

### 步骤3：信息整合
```
将各轮输出整合成一份完整的"网文蒸馏档案"：
- 统一格式
- 消除重复
- 补充缺失的细节
- 标注各部分的重要程度（★★★ 很重要 / ★★ 重要 / ★ 参考）
```

### 步骤4：验证与微调
```
检查蒸馏内容是否覆盖以下要点：
□ 是否能回答"这部作品讲的是什么"？
□ 是否能列出所有重要人物和他们的故事？
□ 是否理解了世界设定的核心规则？
□ 是否掌握了作品的文笔风格和情感基调？
□ 是否知道如何用原作的语气继续写下去？

如果有缺漏，针对性地加强第二阶段深挖。
```

## 📊 输出格式建议

蒸馏完成后，建议按以下结构保存：

```
📁 [网文名称]_蒸馏档案/
  ├─ 📄 基本信息.md
  │   └─ 作品分类、背景、核心概念、风格基调等
  ├─ 👥 人物系统.md
  │   └─ 每个角色的完整档案
  ├─ 🌍 世界设定.md
  │   └─ 地理、组织、规则、文化等
  ├─ 📖 剧情总结.md
  │   └─ 分阶段的故事发展、转折点、伏笔等
  ├─ ✍️ 风格参考.md
  │   └─ 文笔特色、语言习惯、情感表达方式等
  └─ 🚀 续写指南.md
      └─ 逻辑链条、性格底线、设定约束、续写建议等
```

## ⚠️ 重要提示

1. **关于LLM的选择**：
   - GPT-4 / Claude 3.5 Sonnet：准确度最高（推荐）
   - GPT-3.5 Turbo / Claude 3：成本低，但可能遗漏细节
   - 本地Ollama（Mistral、Llama）：免费但速度慢，准确度一般

2. **关于处理字数限制**：
   - 如果LLM的context window限制（如4K），需要分段处理
   - 每次输入时清晰说明"这是第X部分，请关注以下重点"
   - 最后做一次汇总整合

3. **关于数据隐私**：
   - 如果是未发布或非公开的网文，建议使用本地方案
   - 考虑使用开源LLM（Ollama）以确保隐私

4. **关于质量控制**：
   - 第一次蒸馏完成后要自己读一遍，检查是否遗漏关键信息
   - 对于重要角色和情节，可以要求LLM再次深化分析
   - 保留原文的关键台词和描写，不要过度简化

```

## 🎁 快速模板（3分钟快速版）

如果你赶时间，可以用这个简化版：

```
请蒸馏分析这部网文，输出以下内容：

【人物档案】（每个主角200字以内）
- 名字 | 性格 | 外貌 | 能力 | 与主角关系

【故事骨架】（不超过500字）
- 开篇 → 第一个重大转折 → 中间关键事件 → 结尾

【核心设定】（不超过300字）
- 这个世界的特殊之处，必须知道的规则

【文笔风格】（不超过200字）
- 这位作者如何说话、如何描写、情感基调是什么

【续写提示】（3-5个要点）
- 续写时最需要注意的是什么
```

## 💡 使用建议

| 场景 | 推荐做法 |
|:---|:---|
| 想快速理解全貌 | 使用快速模板，一次性获得概览 |
| 要写忠实同人文 | 用完整的两阶段提示词，深度蒸馏 |
| 要改编成其他形式 | 重点深挖世界设定和主题，多次迭代 |
| 想参考某个角色 | 用第二阶段深挖人物分析 |
| 不确定如何下笔 | 用"风格参考"部分先学习原作表达方式 |
